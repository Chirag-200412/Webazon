# Webazon
Webazon is a shopping website just like amazon customers can shop here by one click

# Tools used and setup instructions
1. MongoDB(Database)
- Create database in mongodb named amazon_clone
- Create collections named:
  - orders
  - products
  - users
- open cmd/powershell
  - type "mongod"
- now your server has been started

2. Python(Backend)
- open terminal in the folder which you are using currently
- type app.py
- now your backend has been started

3. HTML,CSS,JAVASCRIPT(Frontend)
- start the live server of index.html
